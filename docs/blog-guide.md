# ピコポッコ ブログ管理ガイド

ブログ記事の作成・編集・削除の手順をまとめたガイドです。

---

## 管理画面へのアクセス

### URL
```
https://picopocco.com/admin.html
```

### ログイン
1. メールアドレスとパスワードを入力
2. 「ログイン」ボタンをクリック

> ログイン情報がわからない場合は、りゅうたろうに確認してください。

---

## 記事を書く

### 1. 新しい記事を作成する

1. 管理画面にログイン
2. **「新しい記事を書く」** ボタンをクリック
3. 以下の項目を入力：

| 項目 | 説明 |
|------|------|
| **タイトル** | 記事のタイトル（例：「こどもの日コンサートのお知らせ」） |
| **カテゴリ** | プルダウンから選択（イベント / 動画 / お知らせ / 日記） |
| **状態** | 「下書き」または「公開」を選択 |
| **サムネイル画像** | 枠をクリックして画像を選択（ブログ一覧に表示される画像） |
| **抜粋** | 一覧ページに表示される短い説明文（1〜2行程度） |
| **本文** | エディタで自由に記事を書く |

### 2. 本文エディタの使い方

本文エリアには、Wordのような書式ツールバーがあります：

| ボタン | 機能 |
|--------|------|
| **H2 / H3** | 見出し（大 / 小） |
| **B** | 太字 |
| *I* | 斜体 |
| <u>U</u> | 下線 |
| ~~S~~ | 取り消し線 |
| 🎨 文字色 | テキストの色を変更 |
| 🎨 背景色 | テキストの背景色を変更 |
| ・リスト | 箇条書き |
| 1. リスト | 番号付きリスト |
| " 引用 | 引用ブロック |
| 🔗 リンク | URLリンクを挿入 |
| 🖼 画像 | 本文中に画像を挿入 |
| 消しゴム | 書式をリセット |

### 3. 画像を挿入する

**サムネイル画像（一覧用）：**
- 「クリックして画像を選択」の枠をクリック
- 画像ファイルを選択
- 自動でアップロード＆プレビュー表示

**本文中の画像：**
- ツールバーの 🖼 画像ボタンをクリック
- 画像ファイルを選択
- 自動でアップロードされ、本文中に挿入される

> 画像は自動でリサイズされるので、大きなファイルでもOKです。

### 4. 保存する

- **「保存する」** ボタンをクリック
- 「保存しました！」と表示されれば完了

> 自動保存機能もあります（30秒ごと）。ブラウザを閉じてしまっても、途中まで書いた内容は残っています。

## 記事を公開する

1. 記事の編集画面で **「状態」** を **「公開」** に変更
2. **「保存する」** をクリック
3. すぐにサイトに反映されます

### 確認方法
- ブログ一覧: https://picopocco.com/blog.html
- トップページのニュース欄にも最新4件が自動表示されます

---

## 記事を編集する

1. 管理画面の記事一覧から、編集したい記事の **「編集」** ボタンをクリック
2. 内容を修正
3. **「保存する」** をクリック

---

## 記事を下書きに戻す

公開中の記事を一時的に非公開にしたい場合：

1. 記事の編集画面を開く
2. **「状態」** を **「下書き」** に変更
3. **「保存する」** をクリック

> 下書き状態の記事は、ブログ一覧やニュース欄には表示されません。

---

## 記事を削除する

1. 記事の編集画面を開く
2. 画面下の **「削除する」** ボタンをクリック
3. 確認ダイアログで **「OK」** をクリック

> 削除した記事は元に戻せません。間違えて消さないように注意してください。

## よくある質問

### Q: 画像がアップロードできない
- 画像ファイルのみ対応しています（JPG, PNG, GIF, WebP）
- 1枚あたり5MBまでです
- 大きすぎる場合は、スマホで撮った写真をそのまま使えば自動リサイズされます

### Q: ログインできない
- メールアドレスとパスワードが正しいか確認してください
- それでもダメな場合は、りゅうたろうに連絡してください

### Q: 記事を公開したのにサイトに表示されない
- ブラウザのキャッシュが原因の場合があります
- ページを再読み込み（スマホなら下に引っ張る）してみてください

### Q: 間違えて公開してしまった
- すぐに編集画面を開いて「状態」を「下書き」に戻せば、サイトから消えます

### Q: 新しいメンバーのアカウントを作りたい
- りゅうたろうに連絡してください（Firebase Consoleから追加します）
